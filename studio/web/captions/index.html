CAPTION EDITOR GUIDE

How to edit captions

  1. Generate narration firstRun “Generate narration” in Studio before editing captions; it creates the timeline.
  2. Locate a captionSelect a cue from a scene or the caption list to locate it in the preview.
  3. Edit and saveEdit text or timing, review the preview, then save captions.json. Narration audio is unchanged.
